## 지금 얘가 어디서 멈춰있냐면

gRPC 이용해서 unary 통신은 구현 완료함.
문제는 stream 통신인데, next 환경에서 stream 받아오려면 websocket 또는 sse 구현 필요
그리하여, envoy proxy로 react - envoy - gRPC Sever 구성을 먼저 구현한 뒤,
시간 남으면 웹소켓이나 sse 구현하고자 함.
